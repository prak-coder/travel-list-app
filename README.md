A app for add items before travel. basically a checklist for important items for travel.. ex passport,dress,phone charger..
Features
--- you can add number and name of items
-- once packed click checkbox items are crossed
---you can remove items
---a small stats at bottom shows no of item packed and to be packed
--you can also sort items by input order,description and packed status
Technologies used
--react
--tailwindcss
--vite
